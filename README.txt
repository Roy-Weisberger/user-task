
Instructions how to run the project on your local computer:
________________________________________________________________
1. Clone or download the project from github: 
	https://github.com/Roy-Weisberger/user-task

2. install additinal libraries (terminal commands):
	-npm i firebase
	-npm install react-scripts --save
	-npm i bootstrap react-bootstrap
	-npm i react-router-dom

3. Add the file .env.local to the project you cloned from Git under user-task
   (the file is not kept in git and will be sent tou you by email)

4. Run npm start
